# Example Catalog

Use these examples as starting points, not as fixed templates.

## Included Sources

- `assets/examples/sequence-service-retry.puml`: synchronous request flow with validation, fallback, and retry outcome
- `assets/examples/sequence-async-worker.puml`: API to queue to worker pipeline with callback-style status progression
- `assets/examples/activity-order-compensation.puml`: compensation-oriented workflow with approval and failure handling
- `assets/examples/component-platform-boundaries.puml`: service boundaries, interfaces, and external systems
- `assets/examples/deployment-edge-platform.puml`: runtime topology across edge, gateway, application, and data nodes

## How To Reuse Them

- copy the closest source file
- replace domain names, actors, and message labels first
- simplify before adding detail
- keep the same style discipline unless the document already has a strong visual system
