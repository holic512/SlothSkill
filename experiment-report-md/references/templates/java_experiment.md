# Java Experiment Template

Use this template for Java coursework or lab projects when Java is clearly the primary implementation language.

Suggested structure:

- Introduction or background
- Requirements analysis
- System architecture and module design
- Data design or database design when present
- Implementation details
- Runtime flow or interface overview
- Testing and issue analysis
- Conclusion

Evidence to prioritize:

- Java package structure
- framework and dependency configuration
- layered architecture
- entity and table definitions
- runtime entrypoints
- tests or execution evidence

Common diagrams:

- module diagram
- sequence diagram
- activity or interaction diagram
- ER diagram when the repository supports it
